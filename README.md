# Testing_Shaping
The repository contains the dataset and the scripts used for the paper "Testing Beliefs, Shaping Diets: A Diachronic Explorative Study of YouTube Users' Interaction in Immigration-Related Videos" submitted to ICWSM conference.

The dataset has been uploaded here for anonimity reasons, upon acceptance it will be made public in a Zenodo repository under Creative Commons Attribution 4.0 International license.
